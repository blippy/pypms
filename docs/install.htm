Installing PyPMS


This text file explains how to install the software for PYPMS


===============================================================================
USERS

This section is for everyone. The source files are in N:\IT\software\python24.

1. Install python-2.4.msi

2. Install pywin32-204.win32-py2.4.exe

3. Add Python to your path:
   1. From the desktop, right-click My Computer and click properties. 
   2. In the System Properties window, click on the Advanced tab. 
   3. In the Advanced section, click the Environment Variables button.
   4. In the section User variables for mark.carter (or whoever you are), 
      either add a new PATH variable, or alter an existing one so that it 
      contains the path c:\python24;
	Source: http://www.computerhope.com/issues/ch000549.htm

4. Make python files executable:
   1. Navigate to the directory M:\Finance\pypms\bin
   2. Right-click: Open With > Choose Program
   3. Browse to C:\Python24\python.exe , and accept it

===============================================================================


DEVELOPERS

This section is only for people who want to develop pypms.

1. Perform installation as for users.

2. Make python files editable: As per Users step 4, but browse to file 
	C:\Python24\Lib\site-packages\pythonwin\pythonwin.exe

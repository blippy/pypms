Developing PyPMS

Hg (Mercurial)

PyPMS uses Mercurial as its source control system. You will need to install it if you want to keep the source under proper control. Typical commands for updating the bin branch from dev: 
cd bin
hg pull .. /dev
hg update

Building an executable

In a DOS shell, run: create-dist.bat . Then manually copy the dist directory over to N:\Project\smithware\pypms
